## Installation

```bash
$ yarn
```

## Running the app

You need to change the `.env.local.example` to `.env.local` and start the [BE](https://github.com/tamdao/swivel-be) first

```bash
# development
$ yarn dev
```

## Test

```bash
# tests
$ yarn start:ci
$ yarn test
```
